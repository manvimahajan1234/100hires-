## Challenges & How I Solved Them
- Git was not available by default on Mac — solved by installing Apple Command Line Developer Tools
- Cursor's UI has been completely redesigned in 2026 — adapted by switching to the Editor Window
- GitHub no longer accepts passwords — solved by generating a Personal Access Token
- Codex requires a paid ChatGPT subscription — used Claude Code and Python scripting instead
- YouTube transcript API had path issues — debugged and resolved using terminal commands
