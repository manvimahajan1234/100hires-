# 100Hires Portfolio Assignment

## Tools Installed
- Cursor IDE (latest version - redesigned agent-based interface)
- Git (via Apple Command Line Developer Tools)
- GitHub account created
- Claude Code for VS Code (by Anthropic) - installed via Extensions marketplace
- Codex by OpenAI - installed via Extensions marketplace

## Steps Completed
1. Installed Cursor IDE from cursor.com
2. Created a GitHub account at github.com
3. Installed Apple Command Line Developer Tools to enable Git
4. Opened the 100hires project folder in Cursor
5. Initialized a Git repository using `git init`
6. Created this README.md file

## Challenges & How I Solved Them
- Git was not available by default on Mac — solved by installing Apple Command Line Developer Tools
- Cursor's UI has been completely redesigned in 2026 — it now uses an agent-based interface instead of the traditional VS Code layout. I adapted by switching to the Editor Window to access the terminal and file explorer.
- The Claude Code and Codex extensions mentioned in the instructions required searching in the new Extensions panel (⌘⇧X)

## Notes
Cursor has been significantly updated and now runs Composer 2.5 as its built-in AI agent, which provides similar functionality to what Claude Code and Codex previously offered as separate extensions.
